# Annotation Tool GUI

A Streamlit web app for row-by-row annotation of Excel files generated by the [python-perturbation-generator](https://github.com/senemogluc/python-perturbation-generator) pipeline.

## What it does

Upload an `.xlsx` file, then label each row as one of three categories:

| Button | Label | Meaning |
|--------|-------|---------|
| ✅ Preserve | `1` | The perturbation preserves the original meaning |
| ❌ Alter | `0` | The perturbation alters the meaning |
| ❓ Not Sure | `-1` | Unclear |

The app skips already-annotated rows and lets you download the annotated file at any time.

## Setup

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Expected file format

Input `.xlsx` files must have:

| Column | Required | Description |
|--------|----------|-------------|
| `source_text` | ✓ | Original sentence |
| `perturbed_text` | ✓ | Modified sentence |
| `bertscore_f1` or `bertscore` | ✓ | Similarity score |
| `perturbation_type` | — | Displayed as metadata |
| `nli_label` | — | Displayed as metadata |
| `confidence_score` | — | Displayed as metadata |
| `annotations` | — | Created automatically if absent |
